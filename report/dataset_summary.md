# Dataset summary

The motchallenge.net was not working, so dataset was downloaded from Google Drive archive, created by collegue `dlcv-final-project-videos.tar.gz`.

The archive contains six MOT-style video sequences:

| Sequence | Frames | GT rows | Unique IDs |
|---|---:|---:|---:|
| KITTI-17 | 145 | 782 | 9 |
| MOT16-09 | 525 | 8830 | 43 |
| MOT16-11 | 900 | 10076 | 80 |
| PETS09-S2L1 | 795 | 4650 | 19 |
| TUD-Campus | 71 | 359 | 8 |
| TUD-Stadtmitte | 179 | 1156 | 10 |

Each sequence contains the following folders:

- `img1/` with video frames
- `gt/` with ground-truth annotations
- `det/` with detection files
